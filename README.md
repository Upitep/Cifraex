# Проект обмена валют Cifraex

## Описание проекта

Проект Cifraex — это REST-сервис для обмена валют, разработанный на ASP.NET Core 6 с использованием Swagger для документации API.

### Тестовое задание

**Задача**: Реализовать на ASP.NET Core 6 Web API сервис обмена валют с использованием Swagger.

**Основные требования**:
- Сервис не использует аутентификацию и авторизацию;
- Возможность создания неограниченного количества пользователей через REST-API;
- Каждый пользователь имеет денежный счет на каждую валюту в системе;
- Возможность изменения баланса указанного пользователя через REST-API;
- Возможность создания неограниченного количества валют через REST-API;
- Система должна поддерживать функционал конвертации суммы из одной валюты в другую по заданному курсу и комиссии.
- В качестве базы данных используется InMemoryDatabase и Entity Framework
- Покрытие модульными тестами

### Работа с сервисом

1. **Создание валюты**:
Для добавления новой валюты отправьте POST-запрос на `/api/Currency` с JSON-телом, содержащим код и описание валюты.

Пример:
```json
{
  "CurrencyCode": "USD",
  "Description": "Доллар США"
}
```

2. **Создание пользователя**:
Чтобы создать пользователя, отправьте POST-запрос на /api/User с именем пользователя в JSON-формате.

Пример:
```json
{
  "Name": "Иван Иванов"
}
```

3. **Обмен валюты**:
Для выполнения обмена валют отправьте POST-запрос на /api/Exchange с необходимыми параметрами.

Пример запроса на обмен:

```json
{
  "UserId": 1,
  "FromCurrency": "USD",
  "ToCurrency": "EUR",
  "Amount": 100,
  "ExchangeRate": 0.85,
  "CommissionRate": 0.05
}
```
